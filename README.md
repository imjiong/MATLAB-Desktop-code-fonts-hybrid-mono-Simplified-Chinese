# 这是什么
这是一个（些）用于 MATLAB 的中的简体中文和英文等宽（mono）的混合字体。个人最喜欢的编程字体, 放到GitHub上方便以后自己下载, 使用. [注]: 我是作者.
# 使用方法
打开 MATLAB 的程序文件夹（例如我的是 ```D:\Program Files\MATLAB\R2024b\sys\java\jre\win64\jre\lib\fonts``` ），复制你要用的字体到这里。再打开 MATLAB ，更换相应字体即可。
